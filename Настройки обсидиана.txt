Настройки:
- Файлы и ссылки:
	Файл открытия по умолчанию - Конкретный файл
	Файл для открытия - Kanban/Задачи по разработке игры
	Место для новых заметок по умолчанию - В той же папке, где расположен файл
	Место для вложенных файлов - В папке, указанной ниже -> Images
	Всегда обновлять внутренние ссылки - вкл.
	Удаленные файлы  - перемещать в корзину Obsidian(.trash)
- Шаблоны:
	Путь к папке с шаблонами - Templates
	Формат даты - DD-MM-YYYY
	Формат Времени - HH:mm
