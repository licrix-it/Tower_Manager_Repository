Начинаем серьезный стаф

Вся суть переменных это коробка с именем. ВСЕ
Считай как у нас коробки с ГМ уезжают. Тут та же хуйня. Создал коробку, дал ей имя, сунул что-то внутрь.

У каждой переменной (коробки) есть свой тип данных. Как если бы у нас были коробки только под телефон или только под чайники.

```C#
// примеры перменных
int age = 20; // Коробка для чисел
string name = "Dimoooon"; // Коробка для строк

// Как объявить переменную:
// тип_данных имя_переменной = значение_переменной


// Знак равно в программировании ничего не сравнивает, это оператор присваивания, он берет то что справа и пихает в коробку слева.
int a = b // берет б и пихает в а

// для сравнивания чисел есть другой оператор (об этом позже)
```

### Основные типы данных:(Остальные можешь изучить сам, посмотрев видосик или статейку почитать)
```C#
int // Тип даннных который может хранить ТОЛЬКО целые числа. Диапазон от -2млрд до + 2млрд (Примерно), точное значение найди в интернете :)
int age = 24;


float // Тип данных для работы с дробными числами(низкая точность). Для этого типа важно не забывать суффикс f в конце. Это указывает проге на тип float, потому что есть еще один для дробей "double".
float PI = 3.14f;


double // Тип данных для дробей, но уже с большей точностью. Занимает чуть больше места в памяти. В остальном такой же как и float. Суфикс уже не надо ставить
double PI = 3.14;


bool // Тип данных для логических значений. Принимает всего 2 состояния true или false
bool iAmGay = false;
bool iAmDNSWorker = true;


string // Тип данных для строковых значений. Длинна строки не имеет значения.
string justFuckingText = "Пишу сюда всякую хуйню потому что могу";


// Нельзя пихать один тип данных в другой
string name = 12; // ошибка. Нельзя в строку пихать число
string name = "12"; // Можно, т.к, все что в кавычках считается строкой, а не числом, и НЕ может использоваться для мат.операций.

int age = "12"; // ошибка. Нельзя в число пихать строку (нельзя чайник пихать в коропку от телефона)
int age = 12; // Все ок

// ну и далее все по аналогии, типы нельзя пихать друг в друга (Пока что :), об этом потом как-нибудь узнаешь)

```

## Объявление и инициализация переменных.
При создании переменных это можно делать в 2 этапа.
1. Объявить переменную `int age;`
2. Присвоить(инициализировать) значение `age = 23;`
А можно все сделать сразу за 1 этап. `int age = 23;`

## Использование переменных
После того как ты создал переменную и дал ей значение, с ней можно уже что-то делать. Складывать, вычитать, выводить в консоль.
Весь доступ к нашей коробке осуществляется по ее имени.

### Что можно делать с числами
Любые числовые типы (int, float, double и др.) Поддерживают математические операции ( + -  *  \  %), а также скобки для приоритета операции, все как в математике
![[Pasted image 20260622082035.png]]
==Важный нюанс: Если ты применяешь операцию на переменной типа int, то и результат будет int. ==
5 / 2 = 2, а не 2.5 потому что работал с целым числом и получил целое число.

Если же нужно поделить нормально чтобы получилась дробь, то и брать изначально надо *дробное* число float или double и делить на float или double
![[Pasted image 20260622082606.png]] Формат записи не сильно важен. Если есть f Это в любом случае float.
Если просто 2.0 это double


Выполнять действия можно(и даже часто нужно) прямо во время инициализации новой переменной.
```C#
int a = 2;
int b = 3;
int c = a + b; // 5
```

### Что можно делать со строками
Строки можно складывать, но не так как числа. Строки по сути просто склеиваются в одну и все. Называется (Конкатенация).
Строки можно складывать с числами, в таком случае число превратиться в строку и также склеится вместе 
```C#
int age = 22;
string str1 = "Меня зовут: ";
string str2 = "Димооон";

Console.WriteLine(str1 + str2); // Меня зовут: Димооон
Console.WriteLine(str1 + str2 + age); //Меня зовут: Димооон22
Console.WirteLine("Можно и так строку написать и что-то добавить" + "добавлю жопку" + 33); //Можно и так строку написать и что-то добавитьдобавлю жопку33. 
//Если забудешь поставить пробелы, то склеятся строки так-же без пробелов.

Console.WirteLine("Меня зовут: " + str2 + ". Мне: " + age + " года."); // Меня зовут: Димоон. Мне: 22 года.
```
