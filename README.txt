Настройки:
- Файлы и ссылки:
	Файл открытия по умолчанию - Конкретный файл
	Файл для открытия - Kanban/Задачи по разработке игры
	Место для новых заметок по умолчанию - В той же папке, где расположен файл
	Место для вложенных файлов - В папке, указанной ниже -> Images
	Всегда обновлять внутренние ссылки - вкл.
	Удаленные файлы  - перемещать в корзину Obsidian(.trash)
- Шаблоны:
	Путь к папке с шаблонами - Templates
	Формат даты - DD-MM-YYYY
	Формат Времени - HH:mm

-----------------------------------------------------------------------------------------------------------------
Правила именования файлов при изменениях: Имя-Файла-Вариант-Фамилия-Дата (Коробки-со-снарягой-Вариант2-Золотов-09.04.26)
!!!НЕ Изменять имена файлов Обсидиана через гитхаб!!! (Слетает автоматическое обновление ссылок в обсидиане). Если надо переименовать файл, переименовать следует в обсидиане и запушить изменение в гитхаб.
!!!Никогда и ничего не удаляем, только в архив

Aseprite:
- Не изменять старые цвета, добавь новый если надо
- Всегда сохранять палитру в конце работы
- Запушить все изменения в гитхаб
